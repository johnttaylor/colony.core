23 /** This template class implements a Local Timer that is context
24  independent and allows for a single context to contain many Timers.
25 
26  A Local timer is defined as timer that has its tick source in the
27  SAME thread. Because the timer methods and callback all occur in the
28  the same thread, the CONTEXT will never receive a timer expired callback
29  AFTER the timer's stop() method has been called.
30 
31  NOTES:
32 
33  o Because the timing source of an individual thread may NOT be a
34  clean divider of the timer duration, the timer duration is taken
35  as the minimum. For example: if the timing source has a resolution
36  of 20msec per count, and the timer duration on the start() timer
37  call is 5 msec, then the timer will expire after the next full count,
38  i.e. after 20msec, not 5msec. IT IS THE APPLICATION'S RESPONSIBILITY
39  TO MANAGE THE RESOLUTION OF THE TIMING SOURCES.
40 
41  Template args:
42  CONTEXT Type of the Class that implements the context for the timer
43  */
