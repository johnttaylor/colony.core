virtual int Cpl::System::Shutdown::Handler::notify ( int  exit_code)
pure virtual

This method is called when the applicaiton is shutting down. The method is passed 'exit_code' which is the exit code provided by the application when called the Shutdown interface. The return value from the handler will be used as the application shutdown exit code. To leave the exit code unalter - return the passed in 'exit_code'.

Notes: 

o The final exit code is the serialized 'sum' of the all of the 
  shutdown handlers.
o The notify() method is called in the thread context that 
  initiated the shutdown.
