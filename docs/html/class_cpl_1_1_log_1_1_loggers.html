Cpl::Log::Loggers Class Reference

Detailed Description

This class provides a collection of pre-defined logger instances. The exact behavior of the loggers below is DETERMINED BY THE APPLICATION and what it platform supports. For example: An application can choose to use the 'Null' logger for all loggers in the collection, which effectively disables all Cpl::Log::Api logging

This method returns a reference to a logger instance tha emphasizes speed of the logging over content, e.g. this logger may dropper lower importance log messages for performance reasons.
