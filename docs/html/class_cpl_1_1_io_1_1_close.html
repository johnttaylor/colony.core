Cpl::Io::Close Class Referenceabstract

Detailed Description

This abstract class defines a close operation that is intended to be used Input and Output streams/files. Since InputOutput streams/files are supported we end up with with 2 close() method when the InputOutput class inherients from Input and Output interfaces. This causes basically a 'diamond' problem. By making the close() it owns interface and a parent class - we can use the 'virtual mechanism' in C++ to ensure that for InputOutput classes there is one and only one close() method.
