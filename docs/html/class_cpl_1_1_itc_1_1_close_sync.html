Cpl::Itc::CloseSync Class Reference

Detailed Description

This partailly concrete class implements the synchronous ITC close call for the CloseApi. The subclass is still required to implement the ITC request() method.

NOTE: This class is derived from OpenSync –>if you support a close(), call, you better have had an open() call.
