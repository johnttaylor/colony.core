Cpl::Io::Null Class Reference

Detailed Description

This concrete class implements a NULL InputOutput stream - that is all input operations return 'End-of-Stream' (i.e. return false) and all output is dropped in the preverbal bit-bucket and goes no where! Note: The write() methods always return true (i.e. no error).
