Detailed Description

template<class SERVER>
class Cpl::Itc::SAP< SERVER >

This concrete template class represents the interface to a ITC Service Access Point (SAP). A SAP 'binds' a Request API with a mailbox interface. In effect a Service is the 'fully-qualified' identifier for posting a message to a specific Request API.
