22 /** This class collects the common functions of a generic
23  server providing a mailbox and a main control loop.
24 
25  The mailbox server is further augmented with the support for Local
26  Timers (see Cpl::Timer::Local). A local timer is started, stopped,
27  and expires all within the context of the thread that the mailbox
28  server is associated with. When the timer expires, it will generate
29  a callback to its CONTEXT instance. The timers and the callbacks
30  (if any timers have expired) are checked/process at 'top of the
31  message loop'. The timers are check before any messages are
32  dispatched/process.
33 
34  The timing source for mailbox server is based on timed-wait operations
35  of a semaphore and should not be considered a deterministic timing
36  source. In addition, the execuation of the callback of a expired timer
37  is subject to the processing that is ongoing in the thread, i.e.if a
38  message takes a relatively long time to be process, it can potentially
39  delay the execution of the timer callback - thus making precision timing
40  intervals even more variable. However, the timer mechanism is guaranteed
41  to NOT expire early with respect to the specified timer duration.
42 
43  */
