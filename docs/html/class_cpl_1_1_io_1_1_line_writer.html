Cpl::Io::LineWriter Class Reference

Detailed Description

This concrete class implements a Line Writer stream using a previously opened Output stream.

NOTE: The LineWriter class does NOT provide any multi-thread mechanisms and/or protections. In addition, using an instance of the AtomicOutputApi interace as the underlying Output stream will NOT make output of the println() calls atomic! The intended way to have an 'Atomic' LineWriter is to use the AtomicOutput callback mechanism (i.e. requestOutputs()) and within the callback create the desired LineWriter using the supplied Output stream.
