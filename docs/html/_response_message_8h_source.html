24 /** This template class represents a client response message, which is posted
25  to the client's mailbox after the corresponding server request message is
26  returned to the client via the server message returnToSender interface. The
27  purpose of this class is to provide a standard model for confirmed
28  asynchronous messages. The SERVER template argument is the type of the
29  server interface that is used by the corresponding server request message.
30  The CLIENT class type is the interface which is invoked by the response
31  message when the client thread invokes the process routine inheritied from
32  the Message class.The CLIENT interface must contain a function named
33  "response", which returns nothing (void) and takes a single argument, which
34  is a reference to this template class type.
35  */
