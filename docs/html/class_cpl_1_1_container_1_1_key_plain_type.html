Detailed Description

template<class DATATYPE>
class Cpl::Container::KeyPlainType< DATATYPE >

This template class is used to generate Key classes for most of the C/C++ primitive data types. This class is NOT intended for general use - it is used create the typedefs that follow.

Key Compare function. Returns <0, 0, >0 if this key is less than, equal, or greater than respectively to the specified key! The actual type of the 'key' is up to the client sub-class that implements this interface. It is the responsibility of the sub-class to correctly define/interpret the data type of the key.
