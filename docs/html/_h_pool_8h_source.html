23 /** This template class defines a concrete Allocator that allocates its block

27 
28  NOTES:
29

32

34

36  than the block size (i.e. sizeof(T)), 0 is returned.
37 
38  4) The class can be deleted. However, it is the responsiblity of the
39  Application to properly clean-up/release ALL outstanding block

42 
43  Template args: class "T" is the type of class to allocated
