Detailed Description

Concrete Block Driver that uses the Host's file system for the persistent storage media.

Linux/Posix and Windoze hosts are supported

Cpl::Io::File::Littlefs::BdFile::BdFile ( const char * fileName)
noexcept

Constructor.

The 'fileName' is the name of the file that will be used as the block device. The file will be created if it does not exist. If the file does exist, it will be opened for read/write access. The string must stay in scope for the life of the object.
