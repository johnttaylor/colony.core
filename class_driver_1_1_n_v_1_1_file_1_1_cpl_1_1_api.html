Driver::NV::File::Cpl::Api Class Reference

This class implements the Non-volatile storage driver using the Cpl::Io::File interfaces. More...

Detailed Description

This class implements the Non-volatile storage driver using the Cpl::Io::File interfaces.

The interface itself is NOT thread safe. It is the responsibility of the users/clients of the driver to handle any threading issues.
