Detailed Description

Concrete Block Driver for a 'traditional' SPI based NOR flash.

The driver is intended to be generic driver with a 24bit address space. That said, the driver was verified using a Winbond W25Q128JV SPI NOR flash device.

#include <BdSpi.h>
